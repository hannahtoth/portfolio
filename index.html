Hi there,

My name is Hannah and I make things.
I studied creative advertising at Ball State University and web design at Eleven Fifty Academy. 
Let me show you some of my work. 






CSS Creature

The CSS Creature is my very first attempt at using HTML and CSS together. For this project I wanted to make something fun while still challenging myself so I chose to create a portrait of my very best friend, Penelope Pasta, who is also my cat. 

Static Web

The Static Web project was my first experience using Bootstrap to design a responsive web page as well as my first time deploying a project on Github Pages.

Api 1

API Challenge 1 was my first attempt at utilizing fetch and DOM manipulation. I used the Studio Ghibli API to fetch and display the names of each of the cats that are featured in the films.  

API 2

API Challenge 2 was my second attempt at utilizing fetch and DOM manipulation. In this project I used NASA's API to display an image of the day when the submit button is clicked.
